# Models

This directory contains the released model files when they are small enough for normal git history.

Current files:

- `best.pt` - YOLO11n PyTorch checkpoint
- `best.onnx` - ONNX export for edge inference
- `SHA256SUMS` - SHA256 checksums for the model files

Future large model files should be uploaded to GitHub Releases instead of committed directly. Release downloads should include checksums where possible.
