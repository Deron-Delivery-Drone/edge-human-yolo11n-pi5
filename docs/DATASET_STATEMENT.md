# Dataset Statement

The raw training dataset is not included in this repository.

This model may have been trained with mixed manually labeled and web-collected images for research and prototyping. This repository shares only the trained model and inference code, not the original images, labels, scraped media, or private dataset manifests.

The individual licenses, rights, and permissions for every original image have not been fully verified. Users should not assume commercial dataset clearance.

If you need a production or commercial model, retrain or fine-tune on a clean dataset with documented rights, licenses, consent, and domain coverage appropriate for your use case.
