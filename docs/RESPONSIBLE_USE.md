# Responsible Use

This model is intended for research, education, prototyping, and edge AI experimentation.

Guidelines:

- Use for research and prototyping.
- Do not use for biometric identity.
- Do not use to target or track specific individuals.
- Respect privacy.
- Follow local laws.
- Test thoroughly before real-world deployment.
- Use human review in safety-related systems.
- Document the deployment context, camera placement, retention policy, and failure modes before field use.

This repository does not make claims that the model, dataset, or dependency stack is commercially cleared.
